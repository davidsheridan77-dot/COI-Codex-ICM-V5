## What is the Codex?
The Codex is COI's permanent source of truth.

It is the constitution, memory, identity, and operating manual of COI -- all in one living system. Every decision flows through it. Every session builds on it.

The Codex lives in GitHub -- accessible from any device, automatically backed up to Dave's PC.

## ICM 4-Layer Architecture

COI follows ICM (Interpretable Context Methodology).
Folder structure is agent architecture. Numbered layers are functional tiers.

| Layer | Folder | Purpose |
|-------|--------|---------|
| L0 | CLAUDE.md | Identity -- always loaded automatically |
| L1 | COI/L1-Routing/ | Navigation, quick-load files, routing |
| L2 | pipeline/*/CONTEXT.md | Stage contracts |
| L3 | COI/L3-Reference/ | Static knowledge -- philosophy, constitution, governance, hardware |
| L4 | COI/L4-Working/ | Memory files, session logs, working artifacts |
