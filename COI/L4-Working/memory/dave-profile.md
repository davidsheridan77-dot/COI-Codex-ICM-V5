# Dave Profile
How Dave thinks. Extracted over time by COI.

## Working Style
- Take it slow. One thing at a time.
- Dave approves before anything gets committed.
- Never leave uncommitted changes without telling Dave.
- Plain conversational language to Dave. Markdown and code only to CC.
- Asks for explanations in plain language, not technical jargon.
- Collaborative approach — "let's refine this" not "do this."
- Values efficiency in interactions — concise commands, direct answers.
- First-principles thinker. No unnecessary complexity.
- Evidence-based decisions — asks for documentation or examples.
- Test not inspect — run it and see if it works.
