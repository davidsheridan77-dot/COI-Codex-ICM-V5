# COI-Codex-ICM-V5

The Codex for COI V5 — persistent memory, context, and operational intelligence.

COI is a personal AI operating system built by Dave Sheridan. V5 is a clean PyQt6 Claude-only foundation. No legacy code. Everything builds from here one phase at a time.

## Structure

| Directory | Purpose |
|-----------|---------|
| COI/ | Core identity, routing, reference, working memory |
| pipeline/ | Build pipeline — intake through deploy |
| scripts/ | Operational scripts |
| ui/ | Desktop interface |
| config/ | Configuration (gitignored) |

## Architecture
ICM — Interpretable Context Methodology. The folder structure is the agent architecture. Files are layered (L0-L4) and loaded on demand based on task context.
